# EXPORT GUIDE

## Exporting Floor Plans as PNG from LibreCAD with 300 DPI Resolution:
1. Open your floor plan in LibreCAD.
2. Go to `File` > `Export`.
3. In the export dialog, select `PNG` from the file type options.
4. Set the desired destination and enter a file name.
5. Before exporting, click on `Settings` to adjust the resolution.
6. Set the DPI to `300`.
7. Click `OK`, and then `Save` to export your drawing as a PNG.
